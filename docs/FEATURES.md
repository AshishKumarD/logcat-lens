# Logcat Lens — Features

## Streaming
- Real-time logcat via ADB with auto device detection
- Pause/Resume/Restart without losing context

## Filtering
- Log level multi-select chips (V/D/I/W/E/F)
- Tag chips with autocomplete
- Package chips with autocomplete
- Full-text search with match counter and navigation
- All filtering is client-side and instant

## Display
- Standard mode with high-performance virtual scrolling
- Compact mode (hides tag, package, PID columns)
- Soft wrap mode with infinite scroll-back history
- Resizable column headers
- Color-coded log levels

## Actions
- Double-click to copy log line
- Export logs to VS Code editor
- Scroll to top/bottom with auto-scroll toggle
